# Glosarium

Istilah teknis spesifik kursus ini. Untuk istilah umum (DNS, tunnel, SSL, dll) lihat Glosarium kursus [oci-domain-cloudflare](https://github.com/1999AZZAR/oci-domain-cloudflare).

---

## C

**CasaOS**
Web-based server operating system. Menyediakan dashboard visual, file manager, dan app store untuk install aplikasi self-host dengan satu klik.

---

## D

**Dashy**
Dashboard startpage yang menampilkan semua service self-host dalam satu halaman. Bisa dikustomisasi dengan icon, kategori, dan status ping.

**Dashboard**
Halaman utama yang menampilkan informasi atau kontrol dalam bentuk visual. CasaOS, Dashy, dan Uptime Kuma semuanya adalah dashboard untuk fungsi berbeda.

---

## P

**Portainer**
Web UI untuk mengelola Docker container. Bisa lihat log, restart, update container tanpa SSH. Lebih advanced dari CasaOS.

---

## U

**Uptime Kuma**
Monitoring tool yang mengecek apakah service online atau offline secara periodik. Mengirim notifikasi (Telegram, Email, dll) jika service mati.
